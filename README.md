## ✨ Features
- **Dynamic 3D Interactions**: Magnetic navigation links and 3D tilt effects on project cards.
- **Interactive Backgrounds**: Grainy noise filter overlays and a custom HTML5 canvas particle network background.
- **Engaging UI Elements**: Custom smart cursor with hover states and dynamic typewriter hero text.
- **Interactive Skills Orbit**: Custom-built animated HTML5 `<canvas>` showcasing my technology stack orbiting a core center.
- **Scroll Animations**: Smooth intersection-observer based reveal animations and statistical counter effects triggered as you scroll.
- **Responsive Layout**: Fluidly adapts to all viewport sizes, from ultrawide monitors to mobile devices.

## 🚀 Installation and Setup

To run this project locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mohanariprasath/Showroom.git
   ```

2. **Navigate into the project directory**
   ```bash
   cd Showroom
   ```

3. **Start a local development server**
   You can use extensions like VS Code's "Live Server" or a simple npx server:
   ```bash
   npx serve ./
   ```

4. **View the application**
   Open your browser and navigate to `http://localhost:3000` (or the port specified by your server).
